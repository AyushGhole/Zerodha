<h1><b>Zerodha </b></h1> 
Developed a full-stack Zerodha-inspired stock trading platform using MERN Stack. Integrated real-time stock market data, user authentication, and an interactive dashboard with live charts, portfolio tracking, and market insights. Designed a responsive UI with React and TailwindCSS for seamless user experience. Backend built with Node.js, Express, and MongoDB to manage transactions and user data. ðŸš€<br><br>

<h1>Key Featrues : </h1> <br>
<ul>

</ul>
